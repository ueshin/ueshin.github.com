Servlet Scopes

Takuya UESHIN (2008年10月29日 14:01) | コメント(1)
Google Guiceは、標準ではServlet関連のスコープを定義していません。

標準で定義されているスコープは、Seasar2で言うところのprototypeとsingletonです。
Guice Servletモジュールを導入すると、requestとsessionのスコープが利用できるようになります。

インストール

Guice ServletモジュールはMaven2レポジトリにデプロイされていないので、ローカルにインストールする必要があります。

ダウンロードページより、guice-1.0.zipをダウンロードします。
適当なディレクトリに展開し、installします。

$ mvn install:install-file \
    -DgroupId=com.google.code.guice \
    -DartifactId=guice-servlet \
    -Dversion=1.0 \
    -Dpackaging=jar \
    -Dfile=/path/to/guice-1.0/guice-servlet-1.0.jar

設定

pom.xml

Guice Servletモジュールを依存に加えます。

pom.xml

<project>

	・・・・

	<dependencies>
		<dependency>
			<groupId>com.google.code.guice</groupId>
			<artifactId>guice</artifactId>
			<version>1.0</version>
		</dependency>
		<dependency>
			<groupId>com.google.code.guice</groupId>
			<artifactId>guice-servlet</artifactId>
			<version>1.0</version>
		</dependency>

	・・・・

	</dependencies>

	・・・・

</project>

GuiceFilter

GuiceにHttpServletRequestとHttpSessionを認識させるため、ServletFilterを設定します。

web.xml

<web-app>

	・・・・

	<filter>
		<filter-name>GuiceFilter</filter-name>
		<filter-class>
			com.google.inject.servlet.GuiceFilter
		</filter-class>
	</filter>

	・・・・

	<filter-mapping>
		<filter-name>GuiceFilter</filter-name>
		<url-pattern>/*</url-pattern>
	</filter-mapping>

	・・・・

</web-app>

ServletModule

Injectorを作成する際に、ServletModuleというModuleを指定します。

    Injector injector = Guice.createInjector(new ServletModule(), ... );

スコープ

Guice Servletモジュールによって追加されるスコープは以下の2つです。

REQUEST

リクエストごとに1つのインスタンスが生成されるスコープです。

SESSION

セッションごとに1つのインスタンスが生成されるスコープです。

Module設定

ModuleでBinderに登録する際に、スコープを指定します。

public class MyModule implements Module {
  public void configure(Binder binder) {
    binder.bind(Service.class)
      .to(ServiceImpl.class)
      .in(RequestScoped.class);
//      .in(SessionScoped.class);
  }
}

インジェクション

通常のインジェクションと同様、@Injectアノテーションを指定することにより依存オブジェクトを抽入することが出来ます。

public class Client {

  private final Service service;

  @Inject
  public Client(Service service) {
    this.service = service;
  }

  public void go() {
    service.go();
  }
}

@RequestParametersアノテーション

リクエストパラメータをMap<String, String[]>オブジェクトとして取得することが出来ます。

public class Client {

  @Inject
  @RequestParameters
  private Map<String, String[]> params;

}
